# Инструкция по размещению на хостинге REG.RU

## Шаг 1: Сборка проекта

Выполните команду для создания production-сборки:

```bash
npm run build
```

Это создаст папку `dist/` с готовыми файлами.

## Шаг 2: Настройка PHP-скрипта

Откройте файл `public/send-contact-form.php` и измените email на свой:

```php
$to_email = 'itc2555888@mail.ru'; // Замените на свой email
```

## Шаг 3: Загрузка на хостинг

1. Войдите в панель управления хостингом REG.RU
2. Откройте файловый менеджер
3. Перейдите в корневую директорию сайта (обычно `public_html/` или `www/`)
4. Загрузите ВСЕ файлы из папки `dist/`:
   - Все `.html`, `.css`, `.js` файлы
   - Папку `assets/`
   - Файл `send-contact-form.php`

## Шаг 4: Проверка работы

1. Откройте ваш сайт в браузере
2. Перейдите на страницу "Контакты"
3. Заполните и отправьте форму
4. Проверьте почту - должно прийти письмо с данными из формы

## Важные примечания

- **PHP должен быть включен** на хостинге (обычно включен по умолчанию на REG.RU)
- **Функция mail()** должна работать на хостинге (стандартно работает на REG.RU)
- Если письма не приходят, проверьте папку "Спам"
- На REG.RU может потребоваться настроить SPF-записи для надежной доставки писем

## Альтернативный вариант (если mail() не работает)

Если встроенная функция PHP mail() не работает, можно использовать SMTP:

1. Установите библиотеку PHPMailer
2. Используйте SMTP вашего почтового провайдера (например, Mail.ru SMTP)
3. Обратитесь в поддержку для подробной настройки
