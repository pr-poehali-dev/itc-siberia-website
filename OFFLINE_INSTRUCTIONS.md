# Инструкция по работе сайта без интернета

## Что изменено

Проект переведен на **Hash Router** - это позволяет открывать сайт напрямую в браузере без веб-сервера.

Теперь все ссылки выглядят так:
- `index.html#/` - главная
- `index.html#/about` - о компании
- `index.html#/services` - услуги
- `index.html#/contacts` - контакты

## Как использовать

### ⚠️ ВАЖНО: Билд требует исправления для автономной работы!

Обычный билд из poehali.dev имеет две проблемы:
1. Абсолютные пути `/assets/` → CORS-ошибка при открытии локально
2. Внешние скрипты cdn.poehali.dev → требуют интернет

**Для работы БЕЗ интернета:**
- Прочитайте файл `OFFLINE_FIX.md` в корне проекта
- Скачайте билд и запустите `node fix-offline.js`
- Или исправьте пути вручную (инструкция в `OFFLINE_FIX.md`)

### Вариант 1: Открыть на компьютере (стандартный билд)
1. Скачайте билд проекта (**Скачать → Скачать билд**)
2. Распакуйте архив
3. Откройте файл `index.html` в любом браузере (двойной клик)
4. ⚠️ Требуется интернет для загрузки внешних скриптов

### Вариант 2: Загрузить на REG.RU
1. Скачайте билд проекта
2. Откройте файл `send-contact-form.php`
3. Измените email на строке 30: `$to_email = 'ваш@email.ru';`
4. Загрузите все файлы из архива в `public_html/` на хостинге
5. Откройте ваш домен в браузере

## Важно

- **Форма обратной связи** имеет два режима работы:
  1. **На хостинге с PHP** - отправка через сервер (основной способ)
  2. **Без PHP** - кнопка "Отправить через почту" откроет ваш почтовый клиент
- При открытии файла напрямую используйте кнопку "Отправить через почту"
- Все остальные функции (навигация, карты, анимации) работают без интернета

## Структура ссылок

Все внутренние ссылки автоматически используют формат с `#`:
- Главная: `#/`
- О компании: `#/about`
- Услуги: `#/services`
- Конкретная услуга: `#/services/angary`
- Контакты: `#/contacts`

Это работает и на хостинге, и при открытии файла напрямую.
