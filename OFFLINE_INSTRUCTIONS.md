# Инструкция по работе сайта без интернета

## Что изменено

Проект переведен на **Hash Router** - это позволяет открывать сайт напрямую в браузере без веб-сервера.

Теперь все ссылки выглядят так:
- `index.html#/` - главная
- `index.html#/about` - о компании
- `index.html#/services` - услуги
- `index.html#/contacts` - контакты

## Как использовать

### Вариант 1: Открыть на компьютере
1. Скачайте билд проекта (**Скачать → Скачать билд**)
2. Распакуйте архив
3. Откройте файл `index.html` в любом браузере (двойной клик)
4. Сайт работает полностью без интернета! ✅

### Вариант 2: Загрузить на REG.RU
1. Скачайте билд проекта
2. Откройте файл `send-contact-form.php`
3. Измените email на строке 30: `$to_email = 'ваш@email.ru';`
4. Загрузите все файлы из архива в `public_html/` на хостинге
5. Откройте ваш домен в браузере

## Важно

- **Форма обратной связи** на странице "Контакты" будет работать только при загрузке на хостинг с PHP
- При открытии файла напрямую форма покажет ошибку - это нормально
- Все остальные функции (навигация, карты, анимации) работают без интернета

## Структура ссылок

Все внутренние ссылки автоматически используют формат с `#`:
- Главная: `#/`
- О компании: `#/about`
- Услуги: `#/services`
- Конкретная услуга: `#/services/angary`
- Контакты: `#/contacts`

Это работает и на хостинге, и при открытии файла напрямую.
